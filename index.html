About

PGrid is a JavaScript data grid plugin for the jQuery JavaScript library, developed by Hunter Perrin as part of 2be. It is designed to provide maximum flexibility, while still being easy to use and easy to implement.

Cross-Browser

PGrid works in all modern browsers and provides a consistent interface. It is tested thoroughly for consistency. No love for legacy browsers.

Touch Friendly

PGrid can be styled specifically for touch screens.

See All Features

Themeable

PGrid uses the jQuery UI CSS library for styling, which means it is fully and easily themeable. Try out some of the readymade themes using the selector in the top right corner of this page.

Completely Open

PGrid is distributed under the GPL, LGPL, and MPL. This triple copyleft licensing model avoids incompatibility with other open source licenses.

 

Did you know?

This is the stable site. You can download code here that's tested and proven to work. However, if you like to live dangerously, you can also check out the development site.

PGrid Features

  • Highly customizable UI.
    • Pagination.
    • Filtering.
    • Sorting.
    • Customizeable toolbar.
    • Column resizing.
    • Column hiding.
    • Selecting.
  • Feature rich API.
    • Hierarchical data.
    • Saving and loading state. (Current page, filter, etc.)
    • Adding and removing items dynamically.
    • Selecting and deselecting items dynamically.
    • Population either statically or dynamically.
    • XHTML 1.0 Strict compliance.
  • Clean, sensible, easy markup.

Installation

Installation is simple. Include jQuery 1.4 (or higher) and jQuery UI (only the CSS is required). Then include the following files:

  • jquery.pgrid.js or jquery.pgrid.min.js (Minified)
  • jquery.pgrid.default.css or jquery.pgrid.touch.css (Touch Screen Styling)
